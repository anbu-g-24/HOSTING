Why should long tasks not block the main thread? => In JavaScript, the main thread is responsible for: Rendering the UI (displaying content) Handling user interactions (clicks, typing, scrolling) Running JavaScript code If a long task blocks the main thread, the following happens: UI Freezes - The page becomes unresponsive. User Actions Fail - Clicks, typing, and scrolling wonnot work. Poor User Experience -The app feels slow or broken.
